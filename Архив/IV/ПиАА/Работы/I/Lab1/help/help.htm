Лабораторная работа №1
Работа с программой

Для старта работы с программой необходимо ввести число N
(сторону квадрата).

После ввода числа, необходимо нажать кнопку "Обработка".

Результат обработки области представляется в виде полосы
загрузки. После достижения 100% программа выведет
изображения областей.

Для больших N программа работает значительно
долгое время, перебирая возможные расположения квадратов.
Для чисел, больших 42, введено ограничение. Алгоритм
бектрекинга не справляется с решением такого уровня задач
в виду время затратности и сложности поставленной задачи.

Возможности:
1)Создание изображений квадратов - решения задачи.
2)Пошаговый режим.
3)Справка.
