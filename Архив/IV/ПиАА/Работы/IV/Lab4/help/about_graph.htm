Работа с графом

Для создания графа необходимо кнопку Генерация или нажать комбинацию клавиш Ctrl+G.

  Возможности программы:

 1) Для создания вершины необходимо произвести двойной клик левой кнопкой мыши по месту где должна быть создана вершина.

 2) Для переименования вершины необходимо произвести двойной клик по вершине в открывшемся окне набрать новое имя вершины и подтвердить свой выбор

 3) Для изменения цвета вершины необходимо произвести по вершине клик правой кнопкой мыши и в открывшемся меню вершины выбрать соответствующее действие

 4) Для удаления вершины необходимо выбрать вершину левой кнопкой мыши далее произвести по вершине клик правой кнопкой мыши и в открывшемся меню вершины выбрать соответствующее действие

 5) Для добавление ребра необходимо выбрать веришну далее зажать клавишу Shift и выбрать вершину, в которой должно кончаться ребро

 6) Для удаления ребра необходимо выбрать ребро левой кнопкой мыши далее правой кнопкой мыши вызвать меню ребра и выбрать соответствующее действие

 7) Для изменения веса ребра необходимо выбрать ребро левой кнопкой мыши далее правой кнопкой мыши вызвать меню ребра и выбрать соответствующее действие

 8) Открытие таблицы смежности. Для того чтобы открыть сохраненную таблицу, необходимо нажать на кнопку "Открыть матрицу из файла" и выбрать текстовый файл. Пример такого файла - example.txt.

 9) Открытие форматированного графа. Для того чтобы открыть форматированный граф, необходимо нажать на кнопку "Открыть форматированный граф" и выбрать файл с расширением graf. Пример такого файла - example.graf. Данный формат файла сохраняет цвет и имена вершин, а также веса ребер.
